# ⚡ K-Electric MTL Unit - Lab Automation System

A professional and modern web-based automation system for **K-Electric**, built with PHP and Bootstrap 5.

## ✨ Features
- **Modern Login:** Glassmorphism UI with KE brand colors.
- **Secure Access:** Admin authentication and session management.
- **Reporting:** Streamlined interface for accessing and generating reports.

## 🛠️ Tech Stack
- **Backend:** PHP 8.x
- **Frontend:** Bootstrap 5, FontAwesome
- **Database:** MySQL
